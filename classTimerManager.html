◆ Update()

void TimerManager::Update ( )
static

Update timers and execute callbacks as needed.

  • This should be called regularly (e.g., in the main loop) to check timers. It checks if the interval has passed for each timer and executes the callback if so. For context-sensitive timers, it also checks if the target window is active before executing.
