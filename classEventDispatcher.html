Detailed Description

Event dispatcher for inter-component communication.

Implements a simple publisher-subscriber pattern for loose coupling between system components. Used primarily for hot-reload notifications.

Manages event subscriptions and dispatch

Allows components to subscribe to events by name and receive callbacks when those events are dispatched. Thread-safe for single producer, single consumer scenarios.
